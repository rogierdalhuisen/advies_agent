## Plan Structure

We provide our Care plans in three different levels, which offer from essential to premium cover:

- **Care Base**
- **Care Enhanced**
- **Care Signature**

Please see the tables in the following pages for a comparison of the three levels and choose the one that suits you.

Under these cover levels, you will find three modules that cover different types of treatments: Core for in-patient and high-cost treatments, then Out-patient and Dental. You must select the type of Core plan you prefer, then you can add either an Out-patient plan or a Dental plan, or both. You can't buy the Out-patient plan and the Dental plan separately, without a Core plan.

Please note that the type of Core plan you select will determine the type of the other modules, e.g. if you choose the Care Base Core Plan, then you must select the Care Base Out-patient Plan, and so on.
